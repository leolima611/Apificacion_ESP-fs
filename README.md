# envio de peticiones a ESP8266 

Este proyecto permite enviar peticiones get y post, adicionalmente tiene un index.html en el que te permite enviar y obtener los datos
utiliza el plugin (SPIFFS/FS) para un sistema mas ordenado en la carpeta data
git de FS: https://github.com/esp8266/arduino-esp8266fs-plugin


## pines utilizados
- ledPin = 16 - gpio16 - D0
- servoPin = 12 - gpio12 -D6
- bateryPin = A0
